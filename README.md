# Playwright Automation Framework
## Overview
This project is a UI Automation Framework developed using Playwright and Java for web application testing.

## Technologies Used
* Java
* Playwright
* Maven
* TestNG
* Page Object Model (POM)
* Git & GitHub

## Framework Features

* Page Object Model Design Pattern
* Reusable Test Components
* Cross Browser Testing
* Data Driven Testing
* Detailed Test Reporting
* Easy Maintenance and Scalability

## Project Structure

```text
src/
├── pages
├── tests
├── utils
└── resources
```

## How to Run

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
mvn clean install
```

Run tests:

```bash
mvn test
```
